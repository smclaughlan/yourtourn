# Endpoints

## Frontend

* "/"
  * Splash page / call to action when not logged in.
  * Search page when logged in.
* "/login"
  * Login form.
* "/register"
  * Registration form.
* "/p/:id"
  * View a player's profile.
* "/t/:id"
  * View a tournament
* "/t/new"
  * Create a new tournament form.
